Vanilla-JS sketches

Each of these loads p5 and p5.transparency from CDN and runs sketch.js directly — no Vite, no bundler. Open this file from the filesystem or let Vite dev serve it; either works.

  • vanilla-js-global — p5 global mode, type-checked via jsconfig + node_modules.
  • vanilla-js-global-standalone — p5 global mode, type-checked against bundled types/ (no node_modules needed for type check).
  • vanilla-js-instance — p5 instance mode, type-checked via jsconfig + node_modules.
  • vanilla-js-instance-standalone — p5 instance mode, type-checked against bundled types/.
